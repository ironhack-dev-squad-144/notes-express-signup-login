# Express Signup and Login

## Setup

```sh
git clone https://github.com/ironhack-dev-squad-144/notes-express-signup-login
cd notes-express-signup-login
npm install
```

Create a `.env` file 
```
PORT=3000
```

## To run the project
```
npm run dev
```
